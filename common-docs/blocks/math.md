# Math

## [Numeric](/types/number) values: 0, 1, 2, ...

```block
0;
1;
2;
```

## Arithmetic binary operation (+, -, *, /)

```block
0+1;
0-1;
1*2;
3/4;
```

## Exponent

```block
4**2;
```

## Absolute value

```block
Math.abs(-5);
```

## Minimum/maximum of two values

```block
Math.min(0, 1);
```

## Random value

```block
Math.random(5);
```
